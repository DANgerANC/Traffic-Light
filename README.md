## О проекте

Проект представляет собой прототип контроллера «Умного перекрёстка» на базе микроконтроллера ESP32. Система предназначена для управления дорожным трафиком по двум осям (Север-Юг и Восток-Запад) с использованием конечного автомата (FSM). 

Ключевая особенность — адаптивность: контроллер способен продлевать зеленый сигнал при наличии потока машин и обеспечивать безопасную фазу для пешеходов. Оператор системы получает полный контроль через HMI-интерфейс (OLED-дисплей + энкодер) с возможностью настройки таймингов и просмотра диагностики.

### Основные функции:
*   Управление светофорами (28 LED: авто + пешеходы).
*   Адаптивное изменение длительности фаз.
*   Режимы: Стандартный, Ночной (мигающий желтый), Аварийный.
*   Интерфейс оператора (вывод статуса и настройка параметров).
*   Самодиагностика через отдельные тестовые пины проверяющие целостность светодиодов.

## MVP (Минимально жизнеспособный продукт)

В рамках текущей итерации (Must Have) будет реализовано:
1.  **Базовая логика светофора (FSM):** Циклическое переключение фаз (Зеленый -> Желтый -> Красный) без конфликтов сигналов.
2.  **Пешеходная фаза:** Реализация безопасного интервала "Всем красный" для перехода пешеходов по запросу.
3.  **Интерфейс оператора:** Отображение текущей фазы и обратного отсчета таймера на OLED дисплее.
4.  **Управление:** временная адаптация к нагрузке по заранее заданным графикам (тестовым).

## Технический стек и оборудование (Wokwi)

Проект разрабатывается на **C++ (Arduino Framework)** для **ESP32**.

**Список компонентов:**
*   **MCU:** ESP32 Dev Kit v1.
*   **Display:** SSD1306 OLED (I2C: SDA=GPIO21, SCL=GPIO22).
*   **RTC:** DS1307 (I2C: 0x68).
*   **Controls:**
    *   Encoder KY-040 (CLK, DT, SW).
    *   Buttons x4 (Пешеход, Поток НС, Поток ВЗ, Авария).
*   **Indication:**
    *   28 LED (Светофоры авто и пешеходные).
    *   Buzzer (Активный зуммер).

## Как запустить симуляцию

Проект настроен для запуска в симуляторе **Wokwi**.

### Вариант 1: VS Code (Рекомендуемый)
1.  Установите расширение **Wokwi for VS Code**.
2.  Клонируйте репозиторий.
3.  Откройте папку проекта в VS Code.
4.  Нажмите `F1` и выберите `Wokwi: Start Simulator`.
5.  Симуляция соберет C++ код и запустит схему, описанную в `diagram.json`.

### Вариант 2: Wokwi Web
1.  Перейдите на сайт [wokwi.com](https://wokwi.com).
2.  Создайте новый проект для ESP32.
3.  Скопируйте содержимое `diagram.json` в вкладку Diagram.
4.  Скопируйте исходный код (`src/main.cpp` или `sketch.ino`) в вкладку Code.
5.  Запустите симуляцию.
